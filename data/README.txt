This folder contains all files of results or their processing, organized in the following subfolders:
`data/raw`: raw data files.

`data/processed`: these are files generated by processing the raw data and these are the ones used in the analysis. 

`data/metadata`: files containing information about the processed data.

`data/results`: files containing analysis output, intermediate data needed to generate figures or tables, or any output generated from processed data sets.
